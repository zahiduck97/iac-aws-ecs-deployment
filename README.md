# Cloud DevOps Microservice Deployment (Terraform + AWS ECS + GitHub Actions)

This project demonstrates full end-to-end CI/CD deployment using:
- Terraform (IaC)
- AWS ECS (Fargate)
- AWS ECR
- AWS VPC / Subnets
- GitHub Actions (CI/CD pipeline)
- Dockerized Node.js backend

✅ 100% Infrastructure as Code  
✅ CI/CD pipeline automated  
✅ Serverless compute with Fargate

**Tech Stack:**
Terraform | AWS | Docker | ECS | ECR | GitHub Actions | CI/CD
